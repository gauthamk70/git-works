> git --version [Command for checking git status]
> git config --list [git configuration list]
> git config --global user.name "username" [command for adding username]
> git config --global user.email "useremail@gmail.com" [command for adding user email]
> git init [initializing git in corrent working directory ]
> git status [list all untracked file]
> git add -A [adding all untracked files into staging area]
> git commit -m "commit message" [adding staged files to local repository]
> git log [command for listing commit history]
> git remote add name remote-path [to add remote repository to local]
